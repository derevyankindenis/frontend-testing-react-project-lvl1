## How to use it

```shell
    $ npx babel-node bin/page-loader --output /var/tmp https://bespoyasov.ru
```
# with logs

```shell
    $ DEBUG=axios,page-loader npx babel-node bin/page-loader --output /var/tmp https://bespoyasov.ru
```

## Run test

```shell
    $ npm test
```

# with logs

```shell
    $ DEBUG=nock.*,axios,page-loader npm test
```
